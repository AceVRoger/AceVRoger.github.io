Welcome to the DOM

Document Object Model: our programming interface for WEB docs.

DOM is set up as a hierarchy.

  • Document Node: entire web page
  • Element Node: HTML tags
  • Attributes Node: Inside Elements
  • Text Node: text inside elements

What's the point?

Allows us to: modify content, change styles, respond to events, add/remove content dynamically.
